# NTU 校園資訊地圖

一個現代化的前端網頁應用，提供台大校園地圖資訊查詢功能。

## 功能特色

### 地圖功能

- **美食**：顯示校園內美食餐廳位置與資訊
- **交通**：
  - 公車：點擊站牌查看各路公車抵達時間
  - Youbike：顯示站點資訊與可用車輛
  - 捷運：顯示最晚時間與最近車站
- **校內**：教室、設施位置與資訊
- **圖書館**：關門時間與目前人數
- **新體健身房**：目前人數與入場券資訊

### 選單功能

- **論壇**：校園論壇功能（開發中）
- **共享課表**：共享課表功能（開發中）
- **活動**：每天登入推播隨機活動，可加入行事曆（開發中）
- **行事曆**：整合 Google Calendar（開發中）
- **登入**：使用者登入功能

## 技術架構

- **HTML5**：頁面結構
- **CSS3**：現代化響應式設計
- **JavaScript (ES6+)**：互動功能
- **Leaflet.js**：地圖顯示與互動

## 使用方式

1. 直接在瀏覽器中開啟 `index.html` 檔案
2. 或使用本地伺服器：

   ```bash
   # 使用 Python
   python -m http.server 8000

   # 使用 Node.js (需安裝 http-server)
   npx http-server
   ```

3. 在瀏覽器中訪問 `http://localhost:8000`

## 操作說明

1. **選擇資訊類別**：點擊左側控制面板的按鈕選擇要查看的資訊類別
2. **查看地圖標記**：選擇類別後，地圖上會顯示相關位置標記
3. **點擊標記或地圖**：點擊標記或地圖上的位置可查看詳細資訊
4. **開啟選單**：點擊右上角選單按鈕開啟功能選單
5. **登入**：從選單中選擇「登入」功能
