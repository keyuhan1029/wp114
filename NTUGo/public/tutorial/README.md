# 教學圖片說明

此資料夾存放教學引導流程使用的圖片。

## 圖片檔案

目前有 5 張示例圖片：
- `slide-1.svg` - 第一張教學圖
- `slide-2.svg` - 第二張教學圖
- `slide-3.svg` - 第三張教學圖
- `slide-4.svg` - 第四張教學圖
- `slide-5.svg` - 第五張教學圖

## 替換圖片

您可以：
1. 將您的圖片命名為 `slide-1.png`, `slide-2.png` 等（支援 .png, .jpg, .svg 格式）
2. 替換對應的檔案
3. 修改 `/src/app/tutorial/page.tsx` 中的圖片路徑（如果需要）

## 圖片建議尺寸

建議圖片尺寸：800x600 像素或比例相同的圖片，以獲得最佳顯示效果。
